# Requirements Phase

Used during **Clarify Vision** objective. Guides PM and BA through gathering, analyzing, documenting, and signing off requirements.

## Steps

| Step | File | Owner |
|------|------|-------|
| 1. Requirements Gathering | `procedures/01-requirements/01-requirements-gathering.md` | PM |
| 2. Requirements Analysis | `procedures/01-requirements/02-requirements-analysis.md` | BA |
| 3. PRD Writing | `procedures/01-requirements/03-prd.md` | PM |
| 4. Review & Sign-off | `procedures/01-requirements/04-requirements-review-and-signoff.md` | PM |

## Key Deliverables

- Structured requirement list (functional, non-functional, business, technical, compliance)
- Gap analysis and dependency graph
- P0/P1 user stories with acceptance criteria
- Signed-off PRD
