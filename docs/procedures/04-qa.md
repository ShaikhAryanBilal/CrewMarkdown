Used during **Verify Quality** objective. Guides QA and Security through validation.

| Step | File | Owner |

| 1. Test Planning | `procedures/04-qa/01-test-planning.md` | QA |
| 2. Test Case Development | `procedures/04-qa/02-test-case-development.md` | QA |

| 4. Non-Functional Testing | `procedures/04-qa/04-non-functional-testing.md` | QA |

- Test plan and test cases
- Bug reports with severity/priority
- Test execution report
- QA sign-off
- Pentest report
