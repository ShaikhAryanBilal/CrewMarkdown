Used during **Design Solution** objective. Guides Architect, UX, and Security through system design.

## Steps

| Step | File | Owner |
|------|------|-------|
| 1. System Architecture | `procedures/02-design/01-system-architecture.md` | Architect |
| 2. Technical Specification | `procedures/02-design/02-technical-specification.md` | Tech Lead |

- Architecture Decision Records (ADRs)
- C4 diagrams (Context, Container, Component)
- API contracts and DB schema
- Wireframes and prototypes
- Threat model with mitigations
