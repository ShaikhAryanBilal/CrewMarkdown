# Light Variant

A compressed 7-step variant of AgentCrew for MVPs, prototypes, and solo development.

## When to Use

| Use Light | Use Full |
|-----------|----------|
| MVPs and prototypes | Production systems |
| Solo development | Team projects |
| Rapid experimentation | Regulated environments |
| Learning/exploration | Enterprise delivery |

## Structure

The light variant is in `.agentcrew/light/` and contains a single `00-router.md` that compresses the full 7 objectives into a streamlined flow.

## Differences from Full

| Aspect | Full | Light |
|--------|------|-------|
| Objectives | 7 with full squads | 7 compressed steps |
| Roles | 16 specialized roles | Reduced role set |
| Ceremony | Security gates, ADRs, debate | Minimal documentation |
| Artifacts | Full PRD, threat model, etc. | Essential artifacts only |
| Overhead | Higher (quality gates) | Lower (speed focused) |

Switch any time. Start with Light → move to Full as complexity grows.
