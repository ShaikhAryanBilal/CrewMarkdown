# Data Engineer

**Owns:** ETL/ELT pipelines, data warehouse/lake, data quality

### Provides
Pipelines, schemas, data catalog, lineage docs, quality checks

### Consumes
Data requirements, source access, storage infrastructure

### Invocation
- **Solo**: *"Data Engineer, build the customer analytics pipeline"*
- **Squad**: With Backend for data API in Build Feature

### Key Skills
ETL/ELT design, data modeling (star, snowflake, vault), data warehouse/lake, stream processing, data quality frameworks, data cataloging, lineage tracking

### Key Procedures
- Pipeline: source analysis → extraction → transformation → loading → validation → monitoring → documentation
