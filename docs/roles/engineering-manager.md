# Engineering Manager

**Owns:** Sprint planning, team process, retrospectives, capacity

### Provides
Sprint plan, capacity allocation, task breakdown, retro report, velocity report

### Consumes
Team capacity, velocity data, priorities from PM

### Invocation
- **Solo**: *"EM, plan the sprint for these features"*
- **Squad**: With PM + Tech Lead for Plan Work objective

### Key Skills
Sprint planning, capacity planning, velocity tracking, retrospectives, Agile/Scrum facilitation, dependency management, burndown/burnup charts

### Key Procedures
- Sprint planning: capacity check → backlog refinement → story pointing → sprint goal → task breakdown → commitment
- Retro: gather data → generate insights → decide actions → follow-up
