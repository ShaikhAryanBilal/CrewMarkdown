Features

Cross-cutting capabilities that enhance any objective — zero-config, skill-level inference, sub-specialization, and 10 special modes.

Zero-Config Entry

Describe what you need in plain English. No CrewMarkdown knowledge required. The agent routes your request, assembles the squad, and executes — all invisible.

./go.ps1
> Build me a landing page for my SaaS product

Progressive disclosure: Start with zero-config, gradually learn role commands as you go.

Skill-Level Inference

CrewMarkdown reads your input and adjusts autonomy automatically. Same system, different experience per user.

ModeAgent AutonomyUser InvolvementExample Input
AutoFull — decides objectives, spawns rolesPassive — approves outcomes"Build my app"
GuidedMedium — proposes plan, seeks inputActive — chooses between options"What stack should I use?"
ExpertLow — assumes domain knowledgeDriving — gives detailed instructions"Architect, design the DB schema"

The agent computes a specificity score from 5 signals: role mention (30%), jargon density (25%), verb specificity (20%), scope granularity (15%), and question format (10%). Mode adapts mid-session based on your follow-ups.

Sub-Specialization

78 sub-specializations across 19 role families. The routing engine scans your request for framework/layer keywords and loads the matching sub-spec contract — falling back to the generic parent if no match found.

Example: "Build a React dashboard" loads the Frontend/UI-React sub-spec. "Deploy to AWS" loads the DevOps/AWS sub-spec.
