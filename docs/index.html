Universal AI Workflow Framework

AI agents with structure, not chaos

Drop .crewmarkdown into any project. Your AI agent handles planning, execution, verification, and delivery — consistently, every time.

Any domain, any skill level. Engineering, marketing, hiring, budgeting — describe what you need. The agent adapts.

You: "Build a landing page" Agent: Plans, routes, assembles squad, executes, verifies, ships

Why CrewMarkdown?

AI agents are capable but lack process. Without a framework, every prompt produces unpredictable results — no plan, no quality checks, no audit trail.

CrewMarkdown fixes this. It's a set of markdown files that give any AI agent a repeatable, production-grade workflow. 40+ specialized roles with contracts, workflows, and quality gates. Zero setup required.

No SaaS. No API keys. No vendor lock. Just plain markdown files any AI agent can read.

Quick Start

For Everyone

# 1. Copy to your project
cp -r .crewmarkdown ./your-project/

# 2. Run the entry point
./go.ps1

Type what you need in plain English. No docs to read. The agent handles everything.

Power User Setup

# 1. Copy to your project
cp -r .crewmarkdown ./your-project/

# 2. Generate codebase map
./.crewmarkdown/scripts/generate-codebase-map.ps1

# 3. Configure your AI agent
./.crewmarkdown/config/setup.ps1        # Windows
./.crewmarkdown/config/setup.sh --all   # Linux/macOS

# 4. Make your first request
#    "PM, write a PRD for user authentication"
#    "Squad (QA, Security), verify the release build"
#    "Build the full platform"

Your AI agent reads .crewmarkdown/00-objectives.md, routes your request to the matching objective, assembles the squad, and executes the workflow with quality gates.
