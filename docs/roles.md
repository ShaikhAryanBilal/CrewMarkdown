# Roles & Responsibilities

AgentCrew defines **16 invocable AI roles**, each with a `contract.md` (inputs/outputs) and `workflow.md` (execution process). Roles can work solo, in squads, or be orchestrated for full lifecycle management.

## Role Categories

| Category | Roles |
|----------|-------|
| **Leadership** | Orchestrator, Product Manager, Tech Lead, Engineering Manager |
| **Analysis** | Business Analyst, Solution Architect, UX Designer |
| **Development** | Frontend Dev, Backend Dev, Mobile Dev, Data Engineer, ML Engineer |
| **Quality** | QA Engineer, Security Engineer |
| **Operations** | DevOps Engineer |
| **Cross-cutting** | Debate Facilitator |

## Invocation Patterns

| Mode | Syntax | Example |
|------|--------|---------|
| **Solo** | `[Role], do [task]` | *"Architect, design the DB"* |
| **Squad** | `Squad ([roles]), achieve [objective]` | *"Squad (PM, BA), clarify the vision"* |
| **Orchestrated** | `Build [feature]` | *"Build the e-commerce platform"* |
