# Contributing to CrewMarkdown

We welcome contributions! To get started:

1. Open an issue to discuss your proposed change.
2. Fork the repo and create a branch.
3. Submit a PR with a clear description of your changes.

For bugs and feature requests, please [open an issue](https://github.com/ShaikhAryanBilal/CrewMarkdown/issues).

This project follows the [CrewMarkdown](.crewmarkdown/00-objectives.md) process for all development work.
