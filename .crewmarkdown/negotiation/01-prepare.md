# Negotiation: Prepare

## Need
- Conflicting interests between parties
- Decision needed that requires agreement

## Instructions

### 1. Identify Parties & Interests
1. Who are the parties involved?
2. What does each party want? (stated positions)
3. Why do they want it? (underlying interests)
4. What's each party's BATNA? (Best Alternative To Negotiated Agreement)

### 2. Define the Issue
1. What exactly needs to be decided?
2. What's the impact of no agreement?
3. What constraints exist (time, budget, policy, technical)?

### 3. Identify Possible Options
1. Brainstorm possible solutions that could satisfy both parties
2. Don't evaluate yet — just list options
3. Identify where interests overlap (potential win-win)

### 4. Set Negotiation Parameters
| Parameter | Description |
|-----------|-------------|
| Must-have | Non-negotiable for each party |
| Nice-to-have | Flexible, can trade |
| Tradeable | What each party can offer in exchange |
| Walk-away | Point where no deal is better than bad deal |

## Done
- Parties, interests, and positions identified
- Issue clearly defined
- Options brainstormed
- Negotiation parameters set
