# Negotiation: Discuss

## Need
- Parties assembled
- Mediator running the session

## Instructions

### 1. Opening (5 min)
1. Mediator sets the stage: goal, process, timebox, ground rules
2. Each party states their position (2 min each, uninterrupted)
3. Mediator restates: "So what I'm hearing is..."

### 2. Explore Interests
1. Move from positions to interests — ask "why is this important to you?"
2. Each party shares what's driving their position
3. Mediator identifies areas of alignment and conflict

### 3. Brainstorm Options
1. Parties suggest possible solutions
2. No judgment or evaluation yet — just generate
3. Build on each other's ideas
4. Aim for 3+ options on the table

### 4. Evaluate Options
1. How well does each option satisfy each party's interests?
2. What trade-offs does each option require?
3. Can options be combined or modified?

## Done
- Positions stated and interests explored
- Options brainstormed without evaluation
- Options evaluated against interests
