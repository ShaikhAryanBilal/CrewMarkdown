# Retrospective: Generate Insights

## Need
- Priority themes identified
- Team ready to explore root causes

## Instructions

### 1. Discuss Each Theme (5-10 min per theme)
For each priority theme:
1. What's behind this? (surface the full picture)
2. What patterns do we see? (is this a one-time thing or recurring?)
3. What systemic factors contribute? (process, tools, culture, structure)

### 2. Identify Root Causes
Use techniques like:
- **5 Whys** — ask "why" five times to get to root cause
- **Fishbone diagram** — categorize causes (people, process, tools, environment)
- **Systemic thinking** — what in our system enables or causes this?

### 3. Capture Insights
1. For each theme, write a clear insight statement:
   - "We keep having X because of Y. If we change Y, we can reduce X."
2. Distinguish symptoms from root causes
3. Avoid blame — always frame as system/process issue

## Done
- Each priority theme discussed in depth
- Root causes identified using structured techniques
- Clear insight statements captured
- Ready to decide on actions
