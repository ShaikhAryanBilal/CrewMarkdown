# Retrospective: Set the Stage

## Need
- Sprint or project phase completed
- Team ready to reflect

## Instructions

### 1. Choose Scope
1. What period are we retrospecting? (sprint, release, project phase, quarter)
2. What's the context? (what happened during this period)

### 2. Set the Tone
1. Emphasize blameless culture — this is about process, not people
2. Set ground rules:
   - Everyone participates
   - All perspectives are valid
   - Focus on what we can control
   - No hierarchy in the retro room

### 3. Pick a Retro Format
| Format | Best For | Duration |
|--------|----------|----------|
| Start/Stop/Continue | Simple, fast, action-oriented | 30-45 min |
| Glad/Sad/Mad | Emotional, captures feelings | 45-60 min |
| 4 L's (Liked/Learned/Lacked/Longed For) | Balanced, reflective | 45-60 min |
| Sailboat (Wind/Anchors/Iceberg/Island) | Systems thinking | 60-75 min |
| Timeline (events + emotions) | Long periods, releases | 60-90 min |
| Mad/Sad/Glad focused | When team is struggling | 45-60 min |

### 4. Assemble Team + Timeline
1. Confirm all team members can participate
2. Set the timebox
3. Designate a Retro Facilitator (not the EM/PM)

## Done
- Scope defined
- Format selected
- Ground rules set
- Team assembled
