# Interview: Plan

## Need
- Need to gather information from a person (user, stakeholder, expert, candidate)
- Interview topic defined

## Instructions

### 1. Define Interview Goals
1. What do you need to learn?
2. Who has that information?
3. How will the information be used?

### 2. Choose Format
| Format | Best For | Duration |
|--------|----------|----------|
| Structured | Specific data, compare across subjects | 30-45 min |
| Semi-structured | Exploratory, user research, discovery | 45-60 min |
| Unstructured | Early exploration, rapport building | 30-60 min |
| Survey | Broad data collection, many subjects | 10-15 min |

### 3. Prepare Questions
1. Write 5-10 core questions
2. For each: what answer are you hoping for?
3. Prepare follow-up probes for each question
4. Order: easy/warm-up first, sensitive/deep later

### 4. Set Logistics
1. Schedule the session
2. Prepare recording/note-taking method
3. Brief the subject on what to expect

## Done
- Goals defined
- Format selected
- Questions prepared with probes
- Logistics confirmed
