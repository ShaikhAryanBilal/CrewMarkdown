# Interview: Conduct

## Need
- Interview planned, questions ready
- Subject present

## Instructions

### 1. Opening (2-3 min)
1. Thank the subject
2. Explain purpose and how data will be used
3. Confirm consent (especially if recording)
4. Set expectations: duration, no wrong answers, they can skip any question

### 2. Core Questions
1. Ask questions as prepared — one at a time
2. Listen more than you speak (80/20 rule)
3. Use active listening — nod, summarize, ask follow-ups
4. Follow interesting threads that emerge (semi-structured)

### 3. Probing Techniques
- **Silence** — wait after they finish, they may add more
- **Echo** — repeat their last phrase as a question
- **Elaboration** — "Can you tell me more about that?"
- **Example** — "Can you give me a specific example?"
- **Comparison** — "How does this compare to...?"

### 4. Closing (2-3 min)
1. Summarize key points heard
2. Ask: "Is there anything else you'd like to add?"
3. Thank them
4. Explain what happens next

## Done
- All core questions asked
- Responses captured (notes or recording)
- Key quotes and observations noted
