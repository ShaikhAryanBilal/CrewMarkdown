# Hackathon: Demo & Judge

## Need
- All teams ready to present
- Judges assembled

## Instructions

### 1. Demo Session
1. Each team gets 5 min demo + 3 min Q&A
2. Demo format:
   - Problem (30 sec)
   - Solution demo (3 min)
   - Tech highlights (1 min)
   - What's next (30 sec)
3. Judges take notes during each demo

### 2. Judging
1. Judges deliberate after all demos
2. Score each team against the criteria
3. Identify winners: 1st place, 2nd place, and any special categories:
   - Best Technical Achievement
   - Most Creative
   - Best UX
   - People's Choice (audience vote)
   - Most Likely to Ship

### 3. Awards
1. Announce winners
2. Celebrate all participants — everyone built something
3. Recognize standout contributions

### 4. What's Next
1. For winning projects: discuss path to production
2. For all projects: archive code, document learnings
3. Collect feedback on the hackathon itself

## Done
- All teams demoed
- Winners selected and announced
- Path forward discussed for top projects
